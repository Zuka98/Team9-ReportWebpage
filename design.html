Design

System Architecture Diagram


Components (left to right, top to bottom)

  1. Share as PDF: User can share their data with a trusted carer as a PDF file
  2. Share as text file: User can share their data with a trusted carer as a text file
  3. Anonymous data sharing: User's data is shared to the nationwide database anonymously
  4. Continuous Deployment: The web app is continously deployed from GitHub and communicates with the Azure MySQL server
  5. GeoJSON Objects: Postcode areas are stored as GeoJSON objects in 2 levels (2nd level stored as properties) to reduce the number of queries required
