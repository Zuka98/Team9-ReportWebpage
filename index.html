Start monitoring your wellbeing using WellWellWell.

WellWellWell is an app that tracks a user’s phone usage statistics to determine their wellbeing following an approved NHS framework - the 5 Ways to Wellbeing. The app tracks the user’s pedometer data (number of steps taken), social media and other telecommunication app usage to generate a wellbeing score, which is indicative of how closely the user is following the NHS framework.
